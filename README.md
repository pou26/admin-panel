# Admin Dashboard


## About

This dashboard utilizes the MERN stack (MongoDB, Express, React, Node.js) to deliver a smooth, interactive experience for analyzing website data. Chart.js enhances data visualization, enabling better insights into trends and patterns.

## Tech Stacks

MongoDB,Express.js,React.js,Node.js,Chart.js

## Run
Run npm i to install node modules in both client and server
Run the application using npm start

access the dashboard at (http://localhost:3000) and server port is 3001
